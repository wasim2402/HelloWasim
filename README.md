# HelloWasim - Personal Portfolio

A modern, high-performance personal portfolio website built with Next.js, TypeScript, and Tailwind CSS. This project showcases my skills, projects, and experience with a sleek, responsive design and interactive elements.

## üöÄ Features

-   **Modern Tech Stack**: Built with Next.js 16 (App Router) and TypeScript for robust performance and type safety.
-   **Responsive Design**: Fully responsive layout ensuring a seamless experience across all devices (Desktop, Tablet, Mobile).
-   **Interactive UI**: Smooth animations and transitions powered by **Framer Motion**.
-   **3D Elements**: Integrated **Three.js** for immersive 3D visual effects.
-   **Dark Theme**: aesthetic dark mode design.
-   **Project Showcase**: Dedicated section to display featured projects with details.
-   **Skills & Expertise**: Visual representation of technical skills.
-   **Contact Form**: Integrated contact section for easy communication.
-   **Admin Panel**: Backend integration for managing content (Projects, Messages).
-   **AI Chat Integration**: Interactive chat component.

## üì¶ Getting Started

Follow these steps to set up the project locally.

### Prerequisites

-   Node.js (v18 or higher recommended)
-   npm or yarn
-   MongoDB instance (local or Atlas)

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/wasim2402/HelloWasim.git
    cd HelloWasim
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Set up environment variables:**

    Create a `.env.local` file in the root directory and add your environment variables (e.g., MongoDB URI, Email credentials).

    ```env
    MONGODB_URI=your_mongodb_connection_string
    # Add other necessary variables
    ```

4.  **Run the development server:**

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
